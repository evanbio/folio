# folio

> A personal reading log — papers and articles, one at a time.

Built with [Quarto](https://quarto.org). No fluff.

## What's inside

Numbered reading notes accumulated daily (001, 002, …).

## Structure

```
folio/
├── paper/        # numbered reading notes (001.qmd, 002.qmd, …)
├── _quarto.yml   # site config
└── index.qmd     # home
```

## Usage

```bash
quarto preview
quarto render
```
